了解 NetApp Trident Protect，它已通過驗證，可用於 OpenShift Virtualization 解決方案的應用程式和持久性儲存管理。

Trident 是一個開源儲存設定器和編排器，由 NetApp 維護，NetApp Trident Protect 可協助您在基於容器的環境（例如 Red Hat OpenShift）中編排和管理持久性資料。
