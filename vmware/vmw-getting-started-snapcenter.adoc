= 了解如何使用適用於 VMware vSphere 的SnapCenter插件備份和還原虛擬機
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
SnapCenter Plug-in for VMware vSphere可對虛擬機器、資料儲存區和 VMDK 檔案實現快速、虛擬機器一致的備份和還原作業。此 VMware 外掛程式與SnapCenter Server 集成，以支援基於應用程式的備份和SnapCenter特定應用程式插件的復原。
