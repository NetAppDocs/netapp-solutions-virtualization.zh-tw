= 在 VI 工作負載域中新增 iSCSI vVols資料儲存作為補充儲存的部署工作流程
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
開始將 iSCI vVols資料儲存配置為 VMware Cloud Foundation (VCF) 虛擬基礎架構 (VI) 工作負載域中的補充儲存。您將建立 SVM 和 LIF、設定 iSCSI 網路、ONTAP tools for VMware vSphere以及設定儲存。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:vmw-vcf-viwld-supp-iscsi-vvols-requirements.html["查看部署要求"]
[role="quick-margin-para"]
查看在 VMware Cloud Foundation VI 工作負載域中部署 iSCSI vVols 的要求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:vmw-vcf-viwld-supp-iscsi-vvols-svm-lifs.html["建立 SVM 和 LIF"]
[role="quick-margin-para"]
為 iSCSI 流量建立具有多個 LIF 的 SVM。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:vmw-vcf-viwld-supp-iscsi-vvols-network.html["設定網路"]
[role="quick-margin-para"]
在 ESXi 主機上設定 iSCSI 網路。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:vmw-vcf-viwld-supp-iscsi-vvols-storage.html["配置儲存"]
[role="quick-margin-para"]
部署並使用ONTAP工具配置儲存。
