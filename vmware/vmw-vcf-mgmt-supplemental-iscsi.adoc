= 在 VCF 管理網域中新增 iSCSI 資料儲存作為補充儲存的部署工作流程
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ../media/


[role="lead"]
開始新增 iSCSI 資料儲存作為 VMware Cloud Foundation (VCF) 管理域的補充儲存。您將設定一個具有用於 iSCSI 的邏輯介面 (LIF) 的儲存虛擬機器 (SVM)，在 ESXi 主機上設定 iSCSI 網絡，ONTAP tools for VMware vSphere，並建立 VMFS 資料儲存區。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-1.png["一"]link:vmw-vcf-mgmt-supplemental-iscsi-requirements.html["查看部署要求"]
[role="quick-margin-para"]
查看將 iSCSI 資料儲存作為補充儲存新增至 VCF 管理域的要求。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-2.png["二"]link:vmw-vcf-mgmt-supplemental-iscsi-svm-lifs.html["建立 SVM 和 LIF"]
[role="quick-margin-para"]
為 iSCSI 流量建立具有多個 LIF 的 SVM。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-3.png["三"]link:vmw-vcf-mgmt-supplemental-iscsi-network.html["設定網路"]
[role="quick-margin-para"]
在 ESXi 主機上設定 iSCSI 網路。

.image:https://raw.githubusercontent.com/NetAppDocs/common/main/media/number-4.png["四"]link:vmw-vcf-mgmt-supplemental-iscsi-storage.html["配置儲存"]
[role="quick-margin-para"]
部署並使用ONTAP工具配置儲存。
